# FsDictCascader

> 字典级联组件
> 支持 el-cascader|a-cascader 组件的参数

## Props

| Prop name | Description                 | Type  | Values | Default |
| --------- | --------------------------- | ----- | ------ | ------- |
| dict      | 字典配置                    |       | -      |         |
| options   | 选项，比 dict.data 优先级高 | array | -      |         |
