
## table【表格配置】

更多参数见：[FsTable](/api/components/crud/crud/fs-table.md)
支持 el-table | a-table的配置

## show
* 说明：显示或隐藏table
* 类型：Boolean
* 默认：`true`

## el-table,a-table参数
* 说明：支持el-table，a-table参数
```json5
{ // table
  border: true,
  bordered: true,
  height: "100%",
  rowKey: "id",
  size: "small",
  stripe: true
}
```

## columns
* 说明：列配置
* 类型：Array

## loading
* 说明：当前是否加载中
* 类型：Boolean
