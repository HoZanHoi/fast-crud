# fs-button
按钮组件，[x]-button的封装

## Props

| Prop name | Description | Type           | Values | Default |
|-----------|-------------| -------------- | ------ | ------- |
| text      | 文本          | string         | -      |         |
| icon      | 图标          | string | -      |       |
| iconRight | 文本右边的图标     | string        | -      |    |
| circle    | 是否圆形        | boolean        | -      | false   |
