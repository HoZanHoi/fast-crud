# fs-icon

> 图标组件
支持多种类型
* 普通类型： ui自带的图标，需要先注册
* iconify： iconify图标，fs-iconify的封装， icon=iconifygroup:iconname格式,如 ion:search
* svg： svg图标，fs-icon-svg的封装， icon=svg:iconname 格式,如 svg:search

## Props

| Prop name | Description     | Type           | Values | Default |
|-----------|-----------------| -------------- | ------ | ------- |
| icon      | 图标名 | string | -      |       |
