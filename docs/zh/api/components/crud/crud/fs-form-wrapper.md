# FsFormWrapper

>表单对话框|抽屉
>暴露方法：
>open(options)

## Props

| Prop name | Description        | Type    | Values | Default   |
|-----------|--------------------|---------| ------ | --------- |
| options   | 表单配置               | object  | -      | {}        |
| inner     | 是否内部打开对话框          | boolean | -      | {}        |
| innerWrapper   | 内部打开对话框时Wrapper的配置 | object  | -      | {}        |
